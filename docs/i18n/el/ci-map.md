# CI Workflow Map

Το αποθετήριο χρησιμοποιεί πλέον ένα ελάχιστο σύνολο CI/CD workflows.

## Ενεργά Workflows

- `.github/workflows/ci-run.yml`
- `.github/workflows/release-build.yml`
- `.github/workflows/pub-release.yml`

## Σημειώσεις

- Το Docker publish είναι απενεργοποιημένο.
- Το release publish ελέγχεται μέσω `RELEASE_AUTHORIZED_ACTORS`.
